Subjects going smoothly

After today, we will have run 10 subjects over the last two days. Things are going smoothly so far. Programming the study in Flash and installing it as an application onto the testing room desktop computers seems to be the most reliable approach. Also, I see better results if we run on the Windows partition versus the Mac side -- I think wireless internet connectivity is spotty unless we're using Windows. So far -- fingers crossed -- we don't have any dropped observations, etc., like we did in some of the pilot data.

We have 14 more signed up through the rest of the week, so I hope it holds. This means I'm likely to wipe out data collection for Project 15 Experiment 2 in February.
