Learning the basics of stats in Python

I have a few free minutes today, so I've been spending some time learning how to use Python for statistical programming. It is not a friendly learning curve so far. I started by installing Anaconda, a package that includes, as far as I can tell, everything you need to get started. Once I did that I got IPython running, which is -- again, as far as I can tell -- a browser-based client for doing stats in Python.

Then I mostly followed a tutorial that assumed I knew way more than I actually did.

A neat feature I discovered is that you can use the IPython Notebook Viewer to share your statistics with others. Click on this link here to see the stats that I ran.

You can see that I conducted a t-test, and a one-way ANOVA! On what, that's a good question, but at least I executed those steps.

So clearly, here's what I need to learn in order to get useful at this:

  • How to take data off of my webservers and get them into IPython

  • Run basic t-tests and ANOVAs with said data

  • Do correlation and regression analyses with said data

None of these things sound too difficult -- rather, it's the conditional analyzing, etc. (e.g., averaging across subjects -- how does one do this?) -- that'll be challenging to learn.

If anyone has any tips, tricks, or good places to get started, feel free to share them.
