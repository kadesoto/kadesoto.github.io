Today my research efforts were spent on obtaining gamma correlations for subjects in my dissertation project. 3-18-14 Data.xlsx contains these numbers. There's not much to look at here, so we can just summarize them by saying that average gamma for remember responses (M = .47, SD = .49) appears numerically greater than average gamma for know responses (M = .26, SD = .33), and those appear greater than gamma for guess responses (M = .08, SD = .28). I haven't run the within-subjects ANOVA yet so I can't tell you whether these differences are significant. Do note that there's a lot of variability here, as illustrated by the standard deviations.
