    • Analyses

      • Project 02 Statistics

      • In the interest of data transparency, openness, etc., I am documenting the analyses I have run on my [02] Experiments 1 and 2 data here. These analyses are being used in a research report of 4,000 words that we are currently writing up for publication. Right now, you will need to e-mail me for the original data files. Everything else, though, will be documented below. Please leave comments, write me, etc. if you have any recommendations or questions about anything I have done here.

      • The manuscript we are currently writing up evolved out of my master's thesis. This document, defended in 2011, included many analyses (and even more were conducted that never made it in), so we had some strong a priori ideas that directed our statistical analyses for the journal article. We also knew we wanted to follow the general organization of our [01] paper, too, which directed our analyses further.

      • We attempt to investigate four things in the manuscript: The average "old" rates (and confidence assigned to these judgments), confidence-accuracy correlations computed in three ways (see Roediger, Wixted, & DeSoto, 2012), effects of output dominance on the false alarm rate, and scatterplots of the 240 items when they were studied and when they were not. I divide these analyses by experiment.

      • Experiment 1

        • Data for these analyses are saved as "02-E1 Analysis.xlsx." The Raw Data tab contains the raw data.

        • "Old" decision rates and confidence to "old" decisions

        • The "Old" Decision Rate tab contains the decision rate data. I constructed a Pivot Table with subject in the rows and type in the columns (1 = target, 2 = related lure, 3 = unrelated lure). Each cell contained a count of the number of "old" responses. I then divided each count by 120 (the total numbers of each item type) to get the proportion.

        • At this juncture I observed that five subjects did not false alarm to any related lures. I ensured that they were not included in the unrelated lure analyses. The means I report below are thus somewhat different from the ones indicated in the Excel document (but the conclusions are the same, although not reported here).

        • The "old" proportions are hit rate = .71 for targets, false alarm rate = .40 for related lures, and false alarm rate = .12 for unrelated lures. The omnibus repeated-measures ANOVA is significant, F(2, 76) = 175.22, p < .001, as are all inter-comparisons.

        • To get the effect sizes, I used an Excel Macro that I created with the help of a presentation by Jason Finley, my office neighbor and postdoc. I just copied and pasted the SPSS output (the Tests of Within-Subjects Effects and Tests of Between-Subjects Effects sections) into this macro to get ω-squared.

        • For this analysis, the ω-squared = 0.71.

        • SPSS output

        • The "Old" Confidence tab contains the confidence data. I copied over the Pivot Table from the "Old" Decision Rate tab and changed the cell values from count to average confidence.

        • The "old" confidence ratings are M = 81 for "old" responses to targets, M = 62 for "old" responses to related lures, and M = 53 for "old" responses to unrelated lures.

        • The omnibus repeated-measures ANOVA is significant, F(2, 76) = 127.94, p < .001, as are all inter-comparisons.

        • Here, ω-squared = 0.42.

        • SPSS output

        • Confidence-accuracy correlations

        • Confidence-accuracy correlations are calculated in three ways: between-subjects, between-events, and within-subjects.

        • Between-subjects

        • For the between-subjects correlations, we create a new Pivot Table and put it on the Between Subjects tab. Here subject goes in the rows, and we will put type as the columns. The contents of each cell are just the confidence. Thus, each cell represents subjects' average confidence to items of each type (or all types, as represented by the Grand Total column). We copy the Pivot Table over and replace average confidence with average accuracy.

        • Then we correlate the corresponding columns. We get r(42) = .68, p < .001 for targets, r(42) = .22, p = .16 for related lures, r = .54 for unrelated lures, and r = .63, p < .001 for all items.

        • Below is the correlation matrix. I only include part of it because we are only really interested in four analyses.

        • SPSS output

        • Between-events

        • For the between-events correlations, we create a new Pivot Table and put it on the Between Events tab. Item goes in the rows, and type is the columns. The contents of each cell are average confidence and average accuracy (in a second table).

        • The all items correlation is complicated, because we need to count some items twice. So we enter the 240 lures, the 240 targets, and the 120 unrelated lures and their corresponding confidences.

        • Then we correlate the corresponding columns. We get r(238) = .61, p < .001 for targets, r(238) = -.26, p < .001 for lures, and r(118) = .38 for unrelated lures. For all items, it is r(598) = .21, p < .001.

        • SPSS output

        • Within-subjects

        • The within-subjects correlation is more complicated and can only be calculated with the help of SPSS. This requires importing the original data file into SPSS. I used one-sample t-tests to compare each mean to zero.

        • The all items (M = .26), target (M = .73), related lure (M = -.21) gammas were all significantly different from zero. The unrelated lure gamma (M = .16) was not.

        • SPSS output

        • Effects of output dominance on false alarm rate

        • This analysis is in the Output Dominance tab of the Excel file. I copied over a Pivot Table, put output dominance in the rows, filtered just to unrelated lures, and put average accuracy in the cells. This gives us the correct rejection rate. The next column over takes (1 - correct rejections) to give us the false alarms.

        • Then we calculate a Pearson correlation between output dominance and false alarm rate, which gives us r(18) = -.90, p < .001.

        • SPSS output

        • For this experiment, we are going to also calculate effects of output dominance on false alarm confidence, too, so we'll copy the Pivot Table and swap accuracy with average confidence. We'll put in another filter so that accuracy is equal to zero.

        • This correlation is r(18) = -.67, p = .002.

        • SPSS output

        • Scatterplots

        • One more thing we want to show are the between-events confidence-accuracy correlations for the 240 items when studied and not studied. This means we'll need to take the between-subjects confidence-accuracy correlations for targets and related lures and plot those. I am not going to show those here.

      • Experiment 2

        • Data for these analyses are saved as "02-E2 Analysis.xlsx."

        • "Old" decision rates and confidence to "old" decisions

        • I basically did the same things here as I did for Experiment 1. The "Old" Decision Rate tab shows the first analyses. Subjects as rows, item type as columns, average accuracy in the cells. We use (1 - correct rejections) and put those in the false alarm rate column, because that's what we're interested in.

        • The "old" decision rates are hit rate = .70 for targets, and false alarm rate = .28 for related lures. The paired-samples t-test is significant, t(25) = 11.18, p < .001.

        • We'll use a different Excel macro to figure out effect size, something else put together with the help of Jason Finley.

        • Effect size here is d = 2.15.

        • SPSS output

        • In the "Old" Confidence tab I do the same thing, except I can just use average confidence in the cells and there's no need to subtract anything. The filter is limited to "old" items.

        • The average "old" confidence ratings are M = 84 for targets and M = 60 for related lures. The paired-samples t-test is significant, t(25) = 8.78, p < .001.

        • Effect size here is d = 2.12.

        • SPSS output

        • Confidence-accuracy correlations

        • Between-subjects

        • In the Between-Subjects tab. Two Pivot Tables, one with confidence averages and one with accuracy averages. Subjects in the rows and item type in the columns. Then we correlate confidence with accuracy for each one.

        • For targets, r(24) = .72, p < .001. For related lures, r(24) = .28, p = .17. For all items, r(24) = .62, p = .001.

        • SPSS output

        • Between-events

        • These correlations are done like the previous ones. New Pivot Tables, word in row, type in columns, correlate. The all items correlations are weird like before, too.

        • For targets, r(238) = .64, p < .001. For related lures, r(238) = -.18, p = .006. For all items, r(478) = .13, p = .006.

        • SPSS output

        • Within-subjects

        • I used one-sample t-tests to compare each mean to zero.

        • The all items (M = .36) and target (M = .79) gammas were both significantly different from zero. The related lure gamma (M = -.14) was not.

        • SPSS output

        • Effects of output dominance on false alarm rate

        • This analysis is in the Output Dominance tab of the Excel file. Output dominance is the rows, average accuracy is in the cells. We take (1 - correct rejections) the next column over to get the false alarm rate for each value.

        • Then we calculate a Pearson correlation, which gives us r(18) = -.92, p < .001.

        • SPSS output

      • Experiment 1 and 2 Comparison

        • One thing I want to do is compare d' in both Experiments 1 and 2. I created a new Excel file ("02 E1-E2 Comparison.xlsx") and copied over the hit and false alarm rates for both experiments. You get d' by taking (normsinv(hit rate) - normsinv(false alarm rate)), which I did for both experiments.

        • Then I did an independent-samples t-test in SPSS.

        • SPSS output

        • I also needed to compare false alarm rates to related lures in both Experiments 1 and 2. This was a last-minute analysis I decided to run, which means it is in the "TEMP" tab. I did an independent-samples t-test in SPSS.

        • Effect size here is d = 0.73.
